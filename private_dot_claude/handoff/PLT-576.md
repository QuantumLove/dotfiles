### Critical Constraints
- **NEVER leak**: `danger_name`, request/response bodies, API keys, auth headers to Datadog
- **ONLY use**: `public_name` (user-facing model names) in spans, logs, metric tags
- **DD_SITE**: `us3.datadoghq.com` everywhere (not the default datadoghq.com)
- **Sentry**: Disable tracing (`SENTRY_TRACES_SAMPLE_RATE=0`) when ddtrace active, keep error capture
- **Memory**: Increase ECS task from 2048 → 3072 MB for sidecars (~300 MB overhead)
- **FluentBit host**: `http-intake.logs.us3.datadoghq.com` for US3
